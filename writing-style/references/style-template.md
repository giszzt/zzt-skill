# 风格档案模板（增量式）

> **使用说明**
> 新建文体档案时复制此模板。本模板基于三层架构设计：
> - **core 层**（`style-core.md`）：跨文体的底层操作系统，已独立存在，文体档案不重复
> - **文体层**（本文件）：记录 core 在该文体中的具体表现 + 该文体独有的规则
> - **样本层**（`samples/`）：原文片段，按需调用
>
> 文件命名：`style-[文体代码].md`（如 `style-wechat.md`、`style-proposal.md`）
